## Project Overview

CS 6120 NLP Final Project: A Retrieval-Augmented Generation (RAG) system that ingests computer vision research papers from arXiv and enables semantic question-answering using an open-source LLM. The system processes, chunks, embeds, and indexes CV papers to provide domain-specific knowledge retrieval and generation capabilities.
